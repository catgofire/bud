# Intro

Let's get started by building a project with bud **in less than 5 minutes**.

## Getting Started

Get started by **creating a new project**.

Create **src/index.js**

```js
console.log('hello world!')
```

## Install

<Tabs
  defaultValue="yarn"
  values={[
    {label: 'yarn', value: 'yarn'},
    {label: 'npm', value: 'npm'},
  ]}>
  <TabItem value="yarn">

    yarn add @roots/bud @roots/bud-cli

  </TabItem>
  <TabItem value="npm">

    npm install @roots/bud @roots/bud-cli --save-dev

  </TabItem>
</Tabs>

## Build your application

<Tabs
  defaultValue="yarn"
  values={[
    {label: 'yarn', value: 'yarn'},
    {label: 'npm', value: 'npm'},
  ]}>
  <TabItem value="yarn">

    yarn bud build

  </TabItem>
  <TabItem value="npm">

    npm run bud build

  </TabItem>
</Tabs>

## Develop your application

Create a **bud.config.json** file in the project root. This file will tell **bud** to serve our js using bud's built-in html template.

```json
{
  "template": true
}
```

Now, start the development server.

```shell
yarn bud dev
```

Your site is now available to view in browser at [localhost:3000](http://localhost:3000).

Open **src/index.js** and make some edits: the site **reloads automatically** and display your changes.
